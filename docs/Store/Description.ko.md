# Microsoft Store 설명 - SRT Probe

## 제품명
SRT Probe - 네트워크 품질 측정 도구

## 설명

SRT Probe는 SRT(Secure Reliable Transport) 프로토콜을 사용하는 네트워크 통신 품질을 측정하는 전문 진단 도구입니다.

라이브 스트리밍 및 실시간 영상 전송 시스템의 사전 검증, 문제 해결, 성능 모니터링에 이상적입니다. 직관적인 GUI를 통해 네트워크 엔지니어부터 비디오 배포 운영자까지 다양한 사용자가 쉽게 SRT 통신 품질을 시각화할 수 있습니다.

주요 기능

- 실시간 모니터링
- 그래프 시각화
- 다중 통신 모드(송신기/수신기, 호출기/수신기, 랑데부)
- 상세 로깅
- 프리셋
- 다국어 지원(영어, 일본어, 중국어, 한국어, 스페인어)

사용 사례

- 비디오 배포 시스템의 사전 배포 검증
- 문제 해결
- 정기 모니터링
- 성능 최적화
- 외부 도구와의 통합
