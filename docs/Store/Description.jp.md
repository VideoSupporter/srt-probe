# Microsoft Store 用の説明 - SRT Probe

## 製品名
SRT Probe - ネットワーク品質測定ツール

## 説明

SRT Probeは、SRT（Secure Reliable Transport）プロトコルを使用したネットワーク通信の品質を測定する、プロフェッショナル向けの診断ツールです。

ライブ配信やリアルタイム映像伝送システムの導入前検証、トラブルシューティング、パフォーマンス監視に最適です。直感的なGUIで、ネットワークエンジニアから映像配信オペレーターまで、幅広いユーザーが簡単にSRT通信の品質を可視化できます。

主な特長

- リアルタイム監視
- グラフ表示
- 複数の通信モード(Sender/Receiver、Caller/Listener、Rendezvous)
- 詳細なログ
- プリセット
- 多言語対応(英語、日本語、中国語、韓国語、スペイン語)

利用シーン

- 映像配信システムの事前検証
- トラブルシューティング
- 定期監視
- パフォーマンス最適化
- 外部ツールとの連携
