# Microsoft Store Description - SRT Probe

## Product Name
SRT Probe - Network Quality Measurement Tool

## Description

SRT Probe is a professional diagnostic tool for measuring network communication quality using the SRT (Secure Reliable Transport) protocol.

Ideal for pre-deployment verification, troubleshooting, and performance monitoring of live streaming and real-time video transmission systems. With an intuitive GUI, users from network engineers to video distribution operators can easily visualize SRT communication quality.

Key Features

- Real-time monitoring
- Graph visualization
- Multiple communication modes (Sender/Receiver, Caller/Listener, Rendezvous)
- Detailed logging
- Presets
- Multi-language support (English, Japanese, Chinese, Korean, Spanish)

Use Cases

- Pre-deployment verification of video distribution systems
- Troubleshooting
- Periodic monitoring
- Performance optimization
- Integration with external tools
