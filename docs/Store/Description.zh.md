# Microsoft Store 说明 - SRT Probe

## 产品名称
SRT Probe - 网络质量测量工具

## 说明

SRT Probe 是一款专业诊断工具，用于测量使用 SRT（Secure Reliable Transport）协议的网络通信质量。

非常适合实时流媒体和实时视频传输系统的部署前验证、故障排除和性能监控。凭借直观的图形用户界面，从网络工程师到视频分发操作员的广泛用户都可以轻松可视化 SRT 通信质量。

主要特点

- 实时监控
- 图形可视化
- 多种通信模式（发送器/接收器、呼叫器/监听器、会合）
- 详细日志
- 预设
- 多语言支持（英语、日语、中文、韩语、西班牙语）

使用场景

- 视频分发系统的部署前验证
- 故障排除
- 定期监控
- 性能优化
- 与外部工具集成
