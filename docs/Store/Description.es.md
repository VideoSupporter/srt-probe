# Descripción de Microsoft Store - SRT Probe

## Nombre del producto
SRT Probe - Herramienta de medición de calidad de red

## Descripción

SRT Probe es una herramienta de diagnóstico profesional para medir la calidad de comunicación de red utilizando el protocolo SRT (Secure Reliable Transport).

Ideal para verificación previa al despliegue, solución de problemas y monitoreo de rendimiento de transmisión en vivo y sistemas de transmisión de video en tiempo real. Con una interfaz gráfica intuitiva, usuarios desde ingenieros de red hasta operadores de distribución de video pueden visualizar fácilmente la calidad de comunicación SRT.

Características principales

- Monitoreo en tiempo real
- Visualización de gráficos
- Múltiples modos de comunicación (Emisor/Receptor, Llamador/Oyente, Encuentro)
- Registro detallado
- Preajustes
- Soporte multilingüe (inglés, japonés, chino, coreano, español)

Casos de uso

- Verificación previa al despliegue de sistemas de distribución de video
- Solución de problemas
- Monitoreo periódico
- Optimización del rendimiento
- Integración con herramientas externas
