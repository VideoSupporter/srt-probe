このアプリケーションは、SRTプロトコルのネットワーク測定を行うネイティブエンジン（srt_probe.exe）を実行するために、runFullTrust機能が必要です。

UWPサンドボックスの制限内では、以下の中核機能が実現できません：

srt_probe.exeプロセスの起動と管理
低レベルのネットワークソケット操作とSRTプロトコル実装
これは、SRTストリーミング接続をテストする必要がある放送エンジニアやネットワーク管理者向けの専門的なネットワーク診断ツールです。


This application requires the runFullTrust capability to execute the native SRT protocol testing engine (srt_probe.exe).

Within UWP sandbox restrictions, the following core functionalities cannot be achieved:

Launching and managing the srt_probe.exe process
Low-level network socket operations and SRT protocol implementation
This is a professional network diagnostic tool designed for broadcast engineers and network administrators who need to test SRT streaming connections.
